# Probability — Bayes' Theorem

Bayes' theorem relates conditional probabilities:

P(A|B) = P(B|A) * P(A) / P(B)

## Worked example

Suppose P(rain) = 0.2 and P(cloudy|rain) = 0.9, P(cloudy|no rain) = 0.3.

P(cloudy) = P(cloudy|rain)P(rain) + P(cloudy|no rain)P(no rain)
         = 0.9 * 0.2 + 0.3 * 0.8 = 0.42

P(rain|cloudy) = 0.9 * 0.2 / 0.42 ≈ 0.429

## Exam tip

Always identify prior, likelihood, and evidence before substituting numbers.
